# ElementMaster Automation Framework

## Project Description

ElementMaster is a Selenium Java Automation Framework built using the Page Object Model (POM) to automate UI testing on the DemoQA practice application.

## Features

- Form automation (text fields, radio buttons, validations)
- Web tables (add, search, delete records)
- Alerts handling (simple, confirm, prompt)
- Frames handling
- Widgets (Date Picker, etc.)
- Screenshot capture on failure
- Extent HTML reports
- FluentWait for dynamic elements

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager
- ExtentReports

## Reports

Reports are generated in `/reports` folder.

## Screenshots

Failure screenshots are stored in `/screenshots`.
